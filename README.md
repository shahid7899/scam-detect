# ScamDetect - Fraud Detection App

**Tagline:** "Fruds ni detect cheyali, mana money ni save cheyali"

## Project Overview

ScamDetect is an innovative web application designed to help users detect and avoid scams. The app uses machine learning and natural language processing to identify potential frauds and provides a platform for users to report suspected scams.

## Features

### Core Functionality
1. **User Registration and Login** - Secure authentication system
2. **Scam Reporting System** - Users can report suspected scams with details
3. **Scam Detection Algorithm** - ML/NLP-based detection of potential scams
4. **Alert System** - Real-time alerts about suspected scams
5. **Education and Awareness** - Tips and information on avoiding scams

### Technical Features
- Modern React frontend with beautiful UI/UX
- Node.js/Express backend with RESTful APIs
- Machine Learning integration for scam detection
- Real-time notifications
- Database management with MongoDB
- Cloud-ready deployment

## Tech Stack

### Frontend

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- MongoDB
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies for both frontend and backend
3. Set up environment variables
4. Start the development servers
